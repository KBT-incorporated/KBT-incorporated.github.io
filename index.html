Purdue Engineering Courses in the Buildings

Have you ever wondered "why is my EE class meeting in Witherell? Who is using our building? How are the various majors distributed across campus?" Wonder no longer.

This visualization shows what classes (grouped by school) are in which buildings. The data shown is incomplete, and some features remain to be implemented, including the ability to filter the data shown by day and time. The code for the treemap as shown is mostly from this source.
